Titanic Data Analysis (Day 15 Project)
Overview

This project analyzes the Titanic dataset to understand survival patterns using Python.

---

What I Did

- Cleaned the dataset
- Handled missing values
- Converted categorical data
- Performed Exploratory Data Analysis (EDA)

---

Key Insights

- Females had a higher survival rate than males
- 1st class passengers survived more
- 3rd class had the lowest survival rate
- Children had better survival chances

---

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
